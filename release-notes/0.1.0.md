- First release of TermTile — a menu-bar app that tiles your terminal windows into an even grid.
- Pick your terminal app (iTerm2 or WezTerm), press **Rearrange now**, and every visible window snaps into columns of two.
- **Check for Updates…** built in, so future versions install themselves.
- Direct download (unsigned for now): on first launch, right-click the app → **Open** to get past the macOS warning.
