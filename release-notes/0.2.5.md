# TermTile 0.2.5

This release adds an optional focus behavior for the manual Rearrange command.

- Added a default off **Bring app forward** option in the Rearrange section. When enabled, TermTile
  still tiles the selected target app first, then asks macOS to bring that target app into focus.
- Kept the existing behavior unchanged when the option is off: Rearrange only moves and resizes
  windows.
- Reorganized the menu around how the app is used: Target selection, Rearrange options, Drag
  behavior, and General app settings now live in separate groups.
- Hardened the foregrounding path so a stale in-flight Rearrange cannot focus an old target app
  after the user changes the target or turns the option off.
- Refined the packaged-app signing checks used by local and release builds.
