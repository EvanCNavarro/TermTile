# TermTile 0.2.6

- Shows an update available signal when Sparkle reports a newer version, with a menu bar indicator and an
  ellipsis attention dot in the menu.
- Keeps the existing **Check for Updates** command as the user-initiated Sparkle update flow.
- Uses a passive update availability check for the indicator instead of parsing the appcast or GitHub
  releases in TermTile.
- Tightens drag-reorder so terminal text selection and screenshot-region drags inside an unchanged
  window do not snap a maximized or focused window back to the grid.
