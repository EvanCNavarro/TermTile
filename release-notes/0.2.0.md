- Added a Gap control, so you can tune the spacing between tiled terminal windows instead of using the fixed default.
- Added a configurable shortcut for Rearrange now, with a recorder in the menu and the active shortcut shown on the primary button.
- Added opt-in drag reorder: turn on Reorder windows on drag, then choose Adaptive, Swap, Shift by column, or Shift by row behavior.
- Improved permissions guidance for Accessibility and Input Monitoring, including clearer recovery copy when a moved or duplicate app breaks an existing grant.
- Added Uninstall TermTile, with scoped cleanup of TermTile-owned data and a reminder to remove the Accessibility permission manually.
- Refreshed the panel into a richer identity card: app icon, version/build, description, GitHub and License links, and a cleaner overflow menu.
- Redesigned the update experience: TermTile now shows its own branded update dialog with the app icon, What's new notes, progress, and install states instead of the plain system update alert.
- Polished the Rearrange now button and menu layout so settings come first, the action sits last, and the shortcut is visible where you use it.
- Under the hood: TermTile now shares MacFaceKit UI components with its companion app RememBar, and the release path is covered by stricter tests.
